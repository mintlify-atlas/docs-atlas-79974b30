Uni ECharts is an Apache ECharts component for uni-app. It lets you render interactive charts across Web, WeChat/Alipay Mini Programs, and APP from a single codebase, with an API nearly identical to [Vue ECharts](https://github.com/ecomfe/vue-echarts) so you can get started quickly without learning a new abstraction.

## Key features

- **Cross-platform** — runs on Web, WeChat Mini Program, Alipay Mini Program, and APP from a single component.
- **Vue 3 Composition API** — full support for `<script setup>` and composables.
- **TypeScript** — ships with `.d.ts` type declarations out of the box.
- **easycom support** — use `<uni-echarts>` without any import when installed via uni-modules.
- **Near-identical API to Vue ECharts** — if you already know Vue ECharts, there is almost nothing new to learn.
- **Free for commercial use** — released under the MIT license.

<Info>
  The npm package `uni-echarts` is currently at version **2.5.1**. The Vite plugin (available since 2.0.0) automates `provideEcharts` injection and optimizeDeps exclusion as of 2.5.0.
</Info>
