Uni ECharts makes it easy to use Apache ECharts in your uni-app projects. With an API that mirrors [Vue ECharts](https://github.com/ecomfe/vue-echarts), you get full cross-platform chart rendering — Web, Mini Programs, and APP — without boilerplate. Install via npm or the uni-modules plugin market and start rendering charts in minutes.

## Why Uni ECharts?

Uni ECharts bridges the gap between Apache ECharts and the uni-app ecosystem. Unlike raw canvas integrations that require manual platform detection, Uni ECharts handles the differences between Web, WeChat Mini Program, Alipay Mini Program, and APP internally — you write your chart option once and it works everywhere.

## Get started in three steps

<Steps>
  <Step title="Install Uni ECharts and ECharts">
    ```bash
    npm install echarts uni-echarts
    ```
  </Step>
  <Step title="Configure the Vite plugin">
    Add the Uni ECharts Vite plugin to your `vite.config.ts` to handle echarts injection automatically.
    ```ts vite.config.ts
    import { UniEcharts } from "uni-echarts/vite";
    import { defineConfig } from "vite";

    export default defineConfig({
      plugins: [UniEcharts()]
    });
    ```
  </Step>
  <Step title="Use the component">
    ```vue
    <template>
      <uni-echarts custom-class="chart" :option="option" />
    </template>

    <script setup lang="ts">
    import { ref } from "vue";

    const option = ref({
      xAxis: { type: "category", data: ["Mon", "Tue", "Wed"] },
      yAxis: { type: "value" },
      series: [{ type: "bar", data: [120, 200, 150] }]
    });
    </script>

    <style>
    .chart { height: 300px; }
    </style>
    ```
  </Step>
</Steps>
